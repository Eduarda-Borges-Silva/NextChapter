# NextChapter – Biblioteca Digital com Integração Kindle

Projeto criado com Create React App e adaptado para as especificações do professor.

## 🛠 Tecnologias usadas

* React (Create React App)
* Material UI (componentes)
* Framer Motion (animações)
* React Router DOM (rotas)
* Google Books API (dados de livros)

## ⚙️ Configuração e execução

1. Navegue até o diretório:
   ```bash
   cd nextchapter
   ```

2. Instale dependências:
   ```bash
   npm install
   ```
3. Inicie o servidor:
   ```bash
   npm start
   ```
4. Acesse http://localhost:3000

## 🔗 Hospedagem

O projeto pode ser publicado em Vercel, Netlify ou Firebase Hosting. Exemplo de link:
```
https://nextchapter-books.vercel.app
```
